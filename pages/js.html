  • Convert non numbers to numbers with the unary +:
    let apples = "2";
    let oranges = "3";
    alert( +apples + +oranges ); // 5

  • Anonymous function: doesnt have a name

  • ReferenceError: object represents an error when a variable that doesn't exist (or hasn't yet been initialized) in the current scope is referenced.

  • SyntaxError: object represents an error when trying to interpret syntactically invalid code. It is thrown when the JavaScript engine encounters tokens or token order that does not conform to the syntax of the language when parsing code.

  • TypeError: object represents an error when an operation could not be performed, typically (but not exclusively) when a value is not of the expected type.

  • Rest Parameter: syntax allows a function to accept an indefinite number of arguments as an array
    Only one rest parameter allowed per function definition and it must be the last parameter.

  • Convert a nodeList into an array with: Array.from() or the spread operator.

  • Style with JS:
    // adds the indicated style rule to the element in the div variable div.style.color = "blue";
    // adds several style rules div.style.cssText = "color: blue; background: white;";
    // adds several style rules div.setAttribute("style", "color: blue; background: white;");

  • Access a style property: // dot notation with kebab case: doesn't work as it attempts to subtract color from div.style.background // equivalent to: div.style.background - color div.style.background-color;
    // dot notation with camelCase: works, accesses the div's background-color style div.style.backgroundColor;
    // bracket notation with kebab-case: also works div.style["background-color"];
    // bracket notation with camelCase: also works div.style["backgroundColor"];
