Home
  • Create a repository, go into the repos folder and copy the ssh key into the terminal like so: git clone git@github.com:USER-NAME/REPOSITORY-NAME.git

  • "git push origin main" to push the project to github(can omit the "origin main" if you are only dealing with one branch or a different remote.)

  • Commit message tips: https://www.theodinproject.com/lessons/foundations-commit-messages

  • Access project through github: go to your github repo > settings > pages > change branch from none to main.(must have an index.html file)

  • Access your project through github: your-github-username.github.io/your-github-repo-name
